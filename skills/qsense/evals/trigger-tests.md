# QSense Skill 触发测试

## 应该触发

1. "帮我看看这张截图里有什么" → 图片识别
2. "这个图片里的文字是什么" → OCR
3. "描述一下这张照片" → 图片描述
4. "把这段录音转成文字" → 音频转录
5. "这个视频讲了什么内容" → 视频理解
6. "对比这两张图的差异" → 多图对比
7. "看看这个 UI 截图有什么问题" → 截图分析
8. "听一下这段音频，总结要点" → 音频理解
9. "这个报错截图是什么意思" → 截图 + 错误分析
10. "分析这段监控视频" → 视频分析

## 不应该触发

1. "帮我写一个图片处理脚本" → 写代码，不是感知
2. "推荐一个视频编辑软件" → 推荐工具
3. "怎么用 ffmpeg 转换视频格式" → 工具使用教程
4. "帮我画一张流程图" → 生成图片，不是理解图片
5. "配置一下摄像头" → 硬件配置
